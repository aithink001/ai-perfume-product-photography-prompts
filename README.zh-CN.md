## 推荐先复制的 Prompt

```text
生成一张 1:1 香水白底商品主图，主体是[香水名称/瓶型]，瓶身居中完整可见，瓶盖、喷头、标签和玻璃边缘清楚。背景为纯白或浅灰无缝棚拍背景，台面干净，产品下方有自然接触阴影。使用柔和大面积主光和轻微边缘光，突出玻璃反光和液体通透感。产品占画面约 70%，不要添加道具、文字、水印或随机品牌 logo。
```
